The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the Open Weather API to build a data set.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather versus latitude.

  • Humidity
  • Cloudiness
